Code Title Trimester Expected Enrolment c f Workload (Hours) Modifiers

Notes

  • The base constant c reports the estimated number of constant hours required (i.e. irrespective of enrolments). For new courses, the base constant is doubled.

  • The coefficient f reports the relative cost per student. Courses with a factor less than 1.0 are supported by professional staff. Those with a load of greater than 1.0 have no SLP support.

See the workload report for more information on the calculation.
